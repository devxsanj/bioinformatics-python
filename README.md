# 🧬 DNA Sequence Analyzer

This is a beginner bioinformatics project built using Python.


- Calculates DNA sequence length
- Counts number of A, T, G, C bases
- Calculates GC content
- Validates input (only A, T, G, C allowed)

## 🧠 What I Learned
- Basic biological sequence analysis
- Applying Python in real-world problems
- Working with user input and validation

## ▶️ How to Run
1. Clone the repository
2. Run the Python file:
   python dna_basics.py

## 📌 Example Input
ATGCTGACCGATAAA

## 📌 Example Output
Length: 15  
A: 6 (40.0%)  
T: 3 (20.0%)  
G: 3 (20.0%)  
C: 3 (20.0%)  
GC Content: 40.0 %
