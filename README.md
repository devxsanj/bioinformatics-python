# Bioinformatics Toolkit

A Python-based toolkit to analyze and compare DNA sequences.

## Features
- DNA sequence analysis (length, base %, GC content)
- Mutation detection between sequences
- FASTA file reading
- Batch processing of multiple sequences
- Comparison summary with similarity %
- Report generation (text files)

## Tech Used
- Python

## How to Run
```bash
python biotoolkit.py
```
